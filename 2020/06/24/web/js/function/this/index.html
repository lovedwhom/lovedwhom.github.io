3. 如果该函数是一个构造函数，this指针指向一个新对象

this return

如果返回中是对象 this指向就是返回的对象，如果返回值不是一个对象那么this还是指向函数的实例
如果是null this还是指向还是函数实例，因为null比较特殊

4. 在严格模式下的函数调用下，this指向undefined

特殊点
当函数作为回调函数时，需要注意是否隐式绑定了所属对象，例如：当作为setTimeout的参数时，就默认将函数绑定了window对象

5. 闭包的this指向

闭包的执行相当于返回一个函数，然后将这个函数执行，因此和普通函数的直接调用相同

6. setTimeout和setInterval 的this指向

由于setTimeOut 调用的代码运行在与所在函数完全分离的执行环境上(异步队列)，这会导致这些代码中包含的this会指向window

7箭头函数的this

箭头函数不会创建自己的this，它只会从自己的作用域链的上一层继承this
在ES6 块级作用域之前 js只有全局作用域和函数作用域
对象没有作用域 对象的多级嵌套只是对象嵌套，这时候没有作用域链的嵌套
